2025-09-19, 2100h

by emsenn

One thing I want to do is try and make sure I take some time to stabilize my research, as I take time to do other work. Like, hacking on potential MUD engines and semiosis simulators is cool, but I’ll benefit from taking time to do what i can to improve GnoponEmacs, the engine I’m actually using to do my research. One thought I have is, if I can get better at using Org-mode clocking, maybe I could try and budget out some fraction of my time spent working toward improving the system I have.

In order to do that - get better at clocking - I’d need to improve the system, which shows something: I can’t try ideas that I imagine, because the system is underdeveloped.

But, short-term, I think I should just try and take some time when I’ve got good thinking capacity to futz in my notes. Right now, if my brain isn’t dead, I’m doing active research, and then when my brain is dead, I’m doing minimal edits and addition of markup. (i.e. flagging nodes as public).

But like, I’ve been using just “blank node” to create almost /all/ of my nodes that I create as I make them as I write. I could do a lot more without much effort if I bothered to write better capture templates - even just for being able to more easily copy in my Fediverse posts.

I think that’s my biggest hangup right now; it’s a big enough hurdle that I can’t actually think of much else I want to tackle right now.

…Though that brings me to: Huh, I don’t actually have a way to plan things in GnoponEmacs.

And, I’m remembering that I wanted to consider reworking my babbles, so that I can potentially more-easily link them together: be able to spin up a babble for exploring a detail in nuance, or for when topics take a turn. But, also, that might be presumptive and inhibit the stream-of-consciousness I’m doing with them currently; another thought is that I could simple start adding in subtrees to structure babbles, after I’m done.

(Long term, I have an idea to have GnoponEmacs bring up specific nodes with specific edit suggestions, i.e. add/revise links, add headlines, whatever.)

Here, I’m realizing I’ve mentioned a few different ideas, and should probably go add them to the appropriate files.
