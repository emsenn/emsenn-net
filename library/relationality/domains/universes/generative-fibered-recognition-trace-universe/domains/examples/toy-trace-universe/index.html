GFRTU toy Model

2025-12-26 by gpt-5.2-codex

This example instantiates the core GFRTU data on a tiny trace site.

Trace Site

  • Objects: t0t_0, t1t_1
  • Morphisms: idt0\mathrm{id}_{t_0}, idt1\mathrm{id}_{t_1}, n:t0→t1n: t_0 \to t_1
  • Topology: trivial

Recognition Fibers

Both fibers are the chain {⊥<m<⊤}\{\bot < m < \top\} with the standard Heyting structure.

Reindexing H(n)H(n) is the identity.

Stabilizer and Drift

  • σt0=σt1=id\sigma_{t_0} = \sigma_{t_1} = \mathrm{id}
  • Δt0(m)=⊤\Delta_{t_0}(m) = \top, others fixed
  • Δt1=id\Delta_{t_1} = \mathrm{id}

Fixed Fibers

  • Ht0∗={⊥,⊤}H_{t_0}^\ast = \{\bot, \top\}
  • Ht1∗={⊥,m,⊤}H_{t_1}^\ast = \{\bot, m, \top\}

Closure Sequence

See goldens/toy-trace-universe-closure.md for expected U0,U1,U2U_0, U_1, U_2.
