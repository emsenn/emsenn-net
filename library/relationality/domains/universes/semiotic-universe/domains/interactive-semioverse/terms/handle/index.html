Handle

2025-12-31

A handle is a persistent indexical reference that points to a thing in the Interactive Semioverse. In Peirce’s trichotomy, an index is a sign connected to its object by existential relation rather than convention or resemblance. A handle formalizes this: it is not an element of the Heyting algebra HH — it lives outside the semantic domain and participates only through interaction terms. The set of all handles is denoted Handle\mathsf{Handle}.

Each handle has a semantic seed η(τ)\eta(\tau) in HH, which is the only point where the handle touches semantics directly. Everything else — the handle’s interaction surface, handle fragment, and footprint — is mediated through interaction terms and closure. This indirection captures the semiotic principle that an index is never fully described by the meaning it currently conveys — it maintains an existential connection that may generate further meaning through further interaction.

Handles support co-reference: a symmetric relation tau ~ tau’ indicates that two handles refer to the same thing, requiring eta(tau) = eta(tau’). Co-referring handles form a groupoid, and transport along co-reference preserves interaction term interpretation within any fragment containing the relevant seeds.
