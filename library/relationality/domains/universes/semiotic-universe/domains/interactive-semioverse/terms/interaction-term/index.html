Interaction Term

2025-12-31
Defines Interaction Term, interaction terms

An interaction term is a structured semiotic act — an expression built from handles (the things pointed at) and definable operators (the ways of combining what is pointed at), interpreted into the Heyting algebra HH. Interaction terms form the smallest set IxTerm\mathsf{IxTerm} generated by: t::=τ∣k(t1,…,tn)t ::= \tau \mid k(t_1, \dots, t_n), where τ\tau is a handle and kk is a definable operator in Opdef\mathsf{Op}^{\mathrm{def}}.

Interpretation extends recursively: a handle τ\tau interprets as its semantic seed η(τ)\eta(\tau), and an operator application k(t1,…,tn)k(t_1, \dots, t_n) interprets as ⟦k⟧(⟦t1⟧,…,⟦tn⟧)\llbracket k\rrbracket(\llbracket t_1\rrbracket, \dots, \llbracket t_n\rrbracket). Interaction terms are the semiotic acts through which handles produce semantic material in HH — the only way external entities generate content in the Interactive Semioverse.

The finite collection of interaction terms associated with a handle in a given state is its interaction surface. The semantic interpretations of these terms generate the handle fragment, and closing that fragment under the state’s operators produces the footprint. If an interaction term is ill-formed or its application undefined, it interprets as bottom (the failure value) and is marked in the failure semantics.
