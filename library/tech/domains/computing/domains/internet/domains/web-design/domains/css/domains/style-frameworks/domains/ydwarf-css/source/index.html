YDwarf

Introduction

YDwarf is a minimal and cold blue CSS theme created by emsenn.

License and Editorial Information

This document and the source code it contains are released under the MIT License and maintained at https://git.sr.ht/~emsenn/ydwarf

Design Philosophy

The YDwarf theme is based on a client's request for a website which would have a landing page containing a sales-letter, a second sales-letter, and finally a link to a form page.

CSS Rules

All of YDwarf's CSS lives in a single style-sheet, ydwarf.css.

Base Rules

The theme's base rules are those that go with HTML tags directly.
