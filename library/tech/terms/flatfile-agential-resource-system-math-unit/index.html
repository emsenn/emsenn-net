Flatfile Agential Resource System Math Unit

Superseded. This concept has been renamed to FlatfileAgentialResourceSystemMathEntity.

The rename follows the repo-wide switch from “unit” to “entity” for named, globally addressable individuals with persistent identity. A file in math/ has an id:, is addressed by other files, and persists across content changes — that is entity semantics, not value semantics.

See unit for the abstract concept of a value/unit. See entity for the abstract concept of a named addressable individual. See flatfile-agential-resource-system-math-entity for the full specification.
