2026-03-30 A substructural logic where premises are resources consumed by use, with tensor (⊗) and linear implication (⊸) as the fundamental adjoint pair.

Linear logic (Girard 1987) is a logic where premises are resources: using a premise consumes it. In classical logic, a hypothesis can be used zero, one, or many times. In linear logic, each hypothesis is used exactly once unless explicitly marked as reusable.

The fundamental connectives are:

  • Tensor (⊗\otimes): multiplicative conjunction. A⊗BA \otimes B means “I have both AA and BB as resources.” Producing A⊗BA \otimes B requires producing both.
  • Linear implication (⊸\multimap): A⊸BA \multimap B means “consuming one AA produces one BB.”
  • Par (\parr): multiplicative disjunction. Dual to tensor.
  • Bang (!A!A): “as many copies of AA as needed.” Recovers classical logic’s unlimited use.

Tensor and linear implication form an adjunction: ⊗⊣⊸\otimes \dashv \multimap. This is the fundamental adjunction of linear logic, analogous to the residuation a⋅b≤ca \cdot b \leq c iff b≤a→cb \leq a \to c in a residuated lattice.

Jakl (2025) proved the Chu construction Chu(H,0)\mathrm{Chu}(H, 0) over a Heyting algebra HH is the twist construction that produces bilattices. Under this identification, tensor in linear logic corresponds to fusion (⊙\odot) in the twist product, and linear implication corresponds to the twist-product implication (⇒\Rightarrow). Every bilattice-valued predicate system is therefore a model of linear logic where the tracked resources are epistemic states.
