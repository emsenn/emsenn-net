Infinity Category

2025-12-31

An ∞-category is a category-like structure with morphisms in all dimensions: objects, morphisms between objects, 2-morphisms between morphisms, 3-morphisms between 2-morphisms, and so on, with coherent composition at every level. Where an ordinary category has only objects and arrows, an ∞-category has arrows between arrows at every height.

Several equivalent models make this idea precise. A quasi-category (or weak Kan complex) is a simplicial set with inner horn fillers. A Segal category or complete Segal space uses simplicial spaces. A simplicially enriched category uses enrichment over simplicial sets. These models are all equivalent in a precise sense and form the basis of higher category theory.

An (∞,1)-category is an ∞-category where all k-morphisms for k > 1 are invertible — the higher morphisms witness homotopies rather than genuinely directed transformations. Most of the ∞-categories arising in practice (spaces, spectra, derived categories, sheaf categories) are (∞,1)-categories. An (∞,1)-topos is an (∞,1)-category satisfying descent conditions that make it a context for homotopy-coherent logic and geometry.
