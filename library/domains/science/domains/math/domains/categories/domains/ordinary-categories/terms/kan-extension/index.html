Kan Extension

2026-03-02

A Kan extension is the universal way to extend a functor along another functor. Given functors F: C → D and K: C → E, the left Kan extension Lan_K(F): E → D is the “best approximation” of F that factors through K.

The left Kan extension Lan_K(F) is characterized by a natural transformation η: F ⇒ Lan_K(F) ∘ K that is universal: for any other functor G: E → D with a natural transformation F ⇒ G ∘ K, there is a unique natural transformation Lan_K(F) ⇒ G making everything commute. The right Kan extension Ran_K(F) satisfies the dual universal property.

Saunders Mac Lane described Kan extensions as the concept from which “all other [categorical] concepts can be derived.” This is not hyperbole. Limits are right Kan extensions along the unique functor to the terminal category. Colimits are left Kan extensions along the same. Adjunctions can be expressed as Kan extensions of the identity functor. Sheafification is a left Kan extension.

When the left Kan extension exists pointwise (computed object-by-object as a colimit), it has an explicit formula:

Lan_K(F)(e) = colim_{(K(c)→e)} F(c)

taken over the comma category (K ↓ e). This computes the extension by gluing together all the values of F that K maps “near” e.

In the semiotic universe formalism, Kan extensions appear in the construction of sheaf semantics: the extension of local semiotic structure to global coherent interpretations is a left Kan extension along the inclusion of local fragments into the global site.
