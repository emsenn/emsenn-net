Zero

2025-12-26

Zero is the number that represents nothing — the quantity you have when you have none of something. It is at once the simplest and most profound of numbers. Every child learns to count from zero (or, historically, from one, because zero took a long time to earn its place). Mathematically, zero is the starting point for the natural numbers, the identity element for addition, and the annihilator for multiplication. It is the only number that is neither positive nor negative. It is a boundary, a balancing point, and a foundation.

In the Peano axioms — the formal foundation for arithmetic — zero is the base element: it is a natural number, and it is not the successor of any natural number. Every other natural number is obtained from zero by applying the successor operation some finite number of times: 1 = S(0), 2 = S(S(0)), 3 = S(S(S(0))), and so on. Zero is where the counting begins.

Zero is the identity element for addition: n+0=0+n=nn + 0 = 0 + n = n for all nn. Adding zero does nothing — it leaves every number unchanged. Zero is the annihilator for multiplication: n×0=0n \times 0 = 0 for all nn. Multiplying by zero collapses everything to zero. These two properties together give zero a dual character: it is inert under addition but destructive under multiplication.

The annihilation property leads directly to the concept of zero divisors in abstract algebra. In the integers, if a×b=0a \times b = 0 then either a=0a = 0 or b=0b = 0 — there is no way for two nonzero integers to multiply to zero. A ring where this holds is called an integral domain. But in other rings, zero divisors can exist: in Z/6Z\mathbb{Z}/6\mathbb{Z}, 2×3=02 \times 3 = 0 even though neither 2 nor 3 is zero. The presence or absence of zero divisors is one of the fundamental structural features of a ring.

Zero was not always recognized as a number. Ancient Greek mathematicians worked without it — their number concept began at one. The Babylonians used a placeholder symbol (two slanted wedges) by around 300 BCE to indicate an empty position in their base-60 numeral system, but they did not treat it as a number in its own right. The breakthrough came from Indian mathematics. Brahmagupta, in Brahmasphutasiddhanta (628 CE), gave explicit rules for arithmetic with zero: a+0=aa + 0 = a, a×0=0a \times 0 = 0, and a−a=0a - a = 0. He also attempted to define division by zero (arriving at some incorrect results, but the attempt itself was remarkable). The concept reached the Islamic world through Muhammad ibn Musa al-Khwarizmi’s Kitab al-Jam’ wal-Tafreeq bil-Hisab al-Hindi (c. 825 CE) and entered Europe through Fibonacci’s Liber Abaci (1202), which introduced the Hindu-Arabic numeral system — including zero — to European commerce and mathematics.

In set theory, zero is represented by the empty set ∅\emptyset — the set with no elements. The cardinality of the empty set is 0. The von Neumann construction builds all the natural numbers from this starting point: 0=∅0 = \emptyset, 1={∅}1 = \{\emptyset\}, 2={∅,{∅}}2 = \{\emptyset, \{\emptyset\}\}, and in general nn is the set of all natural numbers less than nn. Nothing gives rise to everything.
