2026-03-31 A relation between objects and attributes recording which objects have which attributes. The data of a formal context.

An incidence relation II between a set of objects GG and a set of attributes MM records which objects have which attributes.

In the binary case, I⊆G×MI \subseteq G \times M: either object gg has attribute mm or it does not. A checkmark in a spreadsheet cell. A key present or absent in a YAML frontmatter block.

In the many-valued case, I:G×M→LI: G \times M \to L for some value set LL: each object-attribute pair carries a value from LL. When LL is the four-valued bilattice {⊥,t,f,⊤}\{\bot, t, f, \top\}, the incidence records the epistemic state of each assertion — unattested, affirmed, denied, or contested.

An incidence relation is the data of a formal context (G,M,I)(G, M, I). The Galois connection derived from II extracts the concept lattice: the formal concepts and their hierarchy.

A Chu space (Pratt 1999) is a generalization: a triple (A,r,X)(A, r, X) where r:A×X→Lr: A \times X \to L is the evaluation function. A formal context with incidence I:G×M→LI: G \times M \to L is a Chu space with A=GA = G, X=MX = M, r=Ir = I.
