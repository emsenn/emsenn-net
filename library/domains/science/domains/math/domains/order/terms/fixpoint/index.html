2026-03-30 A value unchanged by an operation: f(x) = x. The Knaster-Tarski theorem guarantees least fixpoints exist on complete lattices.

A fixpoint of a function ff is a value xx satisfying f(x)=xf(x) = x. The function maps xx to itself.

The Knaster-Tarski theorem: every monotone function on a complete lattice has a least fixpoint (and a greatest fixpoint). The least fixpoint is the meet of all pre-fixpoints (xx with f(x)≤xf(x) \leq x).

In logic programming on bilattices, Fitting (1991) used the Knaster-Tarski theorem to show that inference operators on bilattice-valued assignments reach a least fixpoint — the closure of the program under its inference rules. The closure makes all derivable knowledge explicit.

On the four-valued bilattice 2⊗2\mathbf{2} \otimes \mathbf{2}, the knowledge ordering ≤k\leq_k has height 2, so fixpoint iteration converges in at most 2n2n steps where nn is the number of assertions. After closure, all in-scope assertions carry values in {t,f,⊤}\{t, f, \top\} — the ⊥\bot (unattested) state has been resolved by inference.
