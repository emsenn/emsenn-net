2026-03-31 A function that preserves order: if a ≤ b then f(a) ≤ f(b). The condition that makes fixpoints exist and closure operators well-behaved.

A function ff between partially ordered sets is monotone if it preserves order:

a≤b  ⟹  f(a)≤f(b)a \leq b \implies f(a) \leq f(b)

Larger inputs give larger (or equal) outputs. The function never reverses the ordering.

Addition of a positive constant is monotone on numbers: if a≤ba \leq b then a+3≤b+3a + 3 \leq b + 3. Meet with a fixed element is monotone on a lattice: if a≤ba \leq b then a∧c≤b∧ca \wedge c \leq b \wedge c. Negation on the integers is NOT monotone: 2≤52 \leq 5 but −2≥−5-2 \geq -5.

Monotonicity is the condition that makes the Knaster-Tarski theorem work: every monotone function on a complete lattice has a least fixpoint. This is why closure reaches a fixpoint — the inference rules on the four-valued bilattice are monotone in the knowledge ordering (Fitting 1991), so iterating them converges.

An antitone (order-reversing) function reverses order: a≤b  ⟹  f(a)≥f(b)a \leq b \implies f(a) \geq f(b). The derivation operators in a Galois connection are antitone: more objects means fewer shared attributes, and more required attributes means fewer qualifying objects.

A bilattice homomorphism must be monotone with respect to both orderings — it preserves both the truth ordering and the knowledge ordering. The interlacing condition on a bilattice says all four operations (meet and join in both orderings) are monotone with respect to both orderings.
