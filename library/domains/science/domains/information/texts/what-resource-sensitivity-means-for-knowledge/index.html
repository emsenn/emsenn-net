What Resource Sensitivity Means for Knowledge

2026-03-31 Linear logic treats premises as resources consumed by use. Applied to knowledge systems, this means assertions have epistemic weight that gets used up when you reason with them.

In classical logic, knowing something is free. If you know AA, you can use AA as many times as you want, in as many arguments as you like, and AA is still there when you are done. Knowledge is unlimited and indestructible.

This is a useful fiction, but it is a fiction.

In practice, using a piece of evidence changes your relationship to it. Citing a source as support for a claim puts weight on that source. If the source turns out to be unreliable, every argument that cited it is weakened. If two sources contradict, using both does not give you twice as much knowledge — it gives you a contradiction.

Linear logic formalizes this. In linear logic, using a premise consumes it. If you know AA, you can use AA once. Using it a second time requires a second copy. The logic tracks what resources you have and what you spend.

Applied to a knowledge system valued in the four-valued bilattice:

Extending combines epistemic content. When you assert “source 1 says PP” (tt for PP) and “source 2 says PP” (another tt for PP), combining them via tensor/fusion (⊙\odot) gives t⊙t=tt \odot t = t: the affirmation is confirmed. But combining “source 1 says PP” (tt) with “source 2 says not-PP” (ff) gives t⊙f=ft \odot f = f: extending an affirmation by a denial collapses to denial. The affirmation was consumed — it did not survive the combination.

Restricting tests against epistemic conditions. Asking “does the evidence for PP survive the constraint of QQ?” uses both the evidence and the constraint. The result depends on the epistemic states of both.

Contradiction consumes evidence. The state ⊤\top (contested) means both affirmation and denial are present. Arriving at ⊤\top consumes the individual affirmation and denial — they are no longer separable. The system knows there is a conflict, but the individual pieces of evidence that produced the conflict have been fused into the contestation.

This is what linear logic’s resource sensitivity means for knowledge: reasoning with evidence uses the evidence. Affirming something and then denying it does not leave you where you started — it leaves you in a state of contestation (⊤\top) that is qualitatively different from ignorance (⊥\bot). The four values track this: ⊥\bot is having nothing, tt and ff are having something, ⊤\top is having too much. Each state is a different kind of epistemic resource, and the residuation tracks how they combine and constrain each other.
