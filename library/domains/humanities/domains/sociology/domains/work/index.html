Work

Work

Work encompasses the practices, structures, and relationships through which humans organize productive activity. This domain includes:

  • Governance: decision-making, authority, and accountability
  • Labor: forms of work, compensation, and obligation
  • Organization: institutional structures and coordination
  • Practice: the actual conduct of work and craft

Key terms

  • Endeavor — sustained, failure-open striving toward an aim
  • Enterprise — aim-generative organized activity
  • Capital — field-specific accumulated resource that generates returns
  • Settlement path — the trajectory from inquiry to process
  • Work — the base concept

Key texts

  • Philosophy of Endeavor — Chisholm, Hornsby, O’Shaughnessy on intentional action
  • The Settlement Path — the organizing principle of a settlement-carrier system
  • Temporality of Endeavors — temporal structure of endeavors
