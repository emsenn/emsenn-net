Recurrence

2026-04-04 The temporal pattern of events that return — what makes a review weekly, a season annual, an obligation generational — distinguished from mere repetition by the novelty each return carries.

Recurrence is the temporal pattern of events that return. A ceremony that happens yearly is a recurrence. A term-limited office whose holder cycles through the community is a recurrence. An obligation that passes to descendants is a recurrence. What makes these recurrences is that the same pattern — the ceremony, the office, the obligation — appears again and again across time.

But recurrence is not repetition. Repetition is the return of identical copies: the same event, unchanged, redundant. Recurrence is the return of the same pattern in changed circumstances. This distinction matters fundamentally. Ilya Prigogine (1997) argued that time has an irreversible arrow — that forward in time means genuinely new states become possible. A spring returns to the same position in the year, but the spring that returns is spring in a world that has aged, where climate has shifted, where seeds are different. The cycle is real; the novelty is also real. The pattern spirals, not circles.

Recurrence operates at multiple temporal scales. Periodic recurrence returns at fixed intervals: daily standups, weekly reviews, annual audits, term-limited elections. Conditional recurrence triggers on events rather than time: a milestone might trigger a retrospective, an error might trigger an investigation, a crisis might trigger a convening. Generational recurrence binds people across lifetimes: a name carries obligation to ancestors; a tradition transmits across generations; the seven-generations principle of Haudenosaunee governance asks whether decisions serve those seven generations forward.

Recurrence is the temporal foundation of governance. Accountability requires recurrence — a review that happens once is an event; a review that recurs produces a pattern of assessment. Term-limited offices, elections that recur, audits that recur — these create the periodic supervision that prevents power from drifting unchecked. The rhythm of return is what governance builds on. H.-G. Gadamer (1960) showed that all understanding occurs within tradition, and tradition is itself a form of recurrence — practices and meanings that return across generations, adapted to new circumstances but continuous with what came before.

The biological world runs on recurrence. Crop rotation works because different crops return in sequence, managing soil and pest cycles. Seeds germinate across multiple years as a bet-hedging strategy against drought or flood. The circadian rhythm returns daily. The menstrual cycle returns monthly. Life itself depends on recurrence — on patterns that return.

Cadence is the operational instantiation of recurrence. Rhythm (in music) is recurrence applied to sound. Tradition is recurrence applied to culture. The concept is general; its domain instances are distinct.
