2025-04-01 Create a glossary that defines every term used in a document in plain language. Use when a document introduces specialized vocabulary.
  • List every non-common term introduced in the document. A term is “non-common” if the default audience (an adult reader with general literacy, no specialized background) wouldn’t know it.
  • For each term, write a one- to three-sentence definition in plain language. Follow the style guide’s term definition conventions: opening definition sentence, then elaboration.
  • Add any symbols or notation used alongside the term. Richard Mayer’s signaling principle: readers comprehend better when essential material is highlighted through structural cues [@mayer2009].
  • Remove circular definitions; each term should be understandable on its own. If term A’s definition requires term B, define B first or include it in the glossary.
  • Use each term consistently throughout the document — don’t rotate synonyms for style. Consistent terminology reduces cognitive load [@sweller1988].
  • Place the glossary adjacent to the document and link to it. If term definitions already exist in the vault, link to those rather than duplicating definitions.
