Predicate set

2026-04-29 The set whose elements are predicates — equivalently, the object set of the predicate category. One of the base parameters of the predication framework, alongside the entity set and the residuated lattice.

A predicate set, denoted Π\Pi, is the object set of a predicate category C\mathcal{C}:

Π:=Ob(C).\Pi := \mathrm{Ob}(\mathcal{C}).

The elements of the predicate set are predicates. The predicate set is one of the base parameters of the predication framework; together with the entity set EE and the residuated lattice L\mathbf{L} from which the bilattice is built, it determines the framework’s basic data.

¶Set vs. category

The predicate set is the underlying set of the predicate category — the bare collection of predicates without the morphisms that relate them. The categorical structure (specialization, converse, the two projections) is layered on top of the set; many definitions and constructions can refer to the set alone when the category structure is not at issue.

When the structure matters — for example, when discussing reindexing along predicate morphisms in a predication fibration — the category C\mathcal{C} is the operative object, and Π\Pi is recovered from it.

¶Cardinality

The predicate set may be finite or infinite. Finite predicate sets correspond to systems with a fixed vocabulary of predicates; infinite predicate sets accommodate systems where predicates can be derived, restricted, or composed through the morphisms of C\mathcal{C}. The framework imposes no specific cardinality requirement on Π\Pi in general; specific theorems may.
