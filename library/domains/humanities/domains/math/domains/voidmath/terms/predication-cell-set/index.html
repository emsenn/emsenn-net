Predication cell set

2026-04-29 The Cartesian product Φ = E × Π × E of an entity set with a predicate set with the entity set again. The set of all predication cells; the index over which a predication valuation assigns values.

The predication cell set, denoted Φ\Phi, is the Cartesian product of an entity set EE, a predicate set Π\Pi, and the entity set EE again:

Φ:=E×Π×E.\Phi := E \times \Pi \times E.

Its elements are the predication cells — triples (s,p,o)(s, p, o) at which a predication valuation assigns a value in the predication fiber algebra ApA_p.

The cell set makes explicit what space the framework’s valuations are functions on: not the entities alone, not the predicates alone, but the full subject-predicate-object index. A valuation is a section over Φ\Phi — assigning a bilattice value to every triple — and it is this sectional structure that the rest of the framework’s machinery (the predicate category, the predication fibration) operates on.

  • Predication fiber algebra — the algebra in which values live, fixed at each predicate slice of Φ\Phi.
