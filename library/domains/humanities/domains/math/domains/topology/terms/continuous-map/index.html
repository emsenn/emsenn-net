Continuous Map

2025-12-26

A continuous map is a function between topological spaces that preserves the structure of nearness — it does not tear, break, or introduce sudden jumps. If two points are close in the source, their images are close in the target. Continuous maps are the structure-preserving functions of topology, just as linear maps are the structure-preserving functions of linear algebra and group homomorphisms are the structure-preserving functions of group theory.

The oldest formulation of continuity is the epsilon-delta definition from calculus. Given metric spaces (X,dX)(X, d_X) and (Y,dY)(Y, d_Y), a function f:X→Yf: X \to Y is continuous at a point xx if for every ε>0\varepsilon > 0 there exists δ>0\delta > 0 such that whenever dX(x,x′)<δd_X(x, x') < \delta, we have dY(f(x),f(x′))<εd_Y(f(x), f(x')) < \varepsilon. In plain language: you can make ff’s output as close as you like to f(x)f(x) by keeping the input close enough to xx. Augustin-Louis Cauchy gave the first rigorous statement of this idea in Cours d’analyse (1821), and Karl Weierstrass sharpened it to the modern epsilon-delta form in his Berlin lectures of the 1860s.

But the epsilon-delta definition uses distances, and topological spaces don’t necessarily have distances. The general definition uses open sets instead. A function f:X→Yf: X \to Y between topological spaces is continuous if for every open set VV in YY, the preimage f−1(V)f^{-1}(V) is open in XX. This single condition replaces epsilon-delta entirely. To see why, think about what an open set around a point yy represents — it is a “neighborhood” of yy, the set of points near yy. Saying f−1(V)f^{-1}(V) is open means that the set of points that map into any neighborhood of yy is itself a neighborhood: nearness is preserved.

This definition is equivalent to several others. A function is continuous if and only if: the preimage of every closed set is closed; or for every point xx and every open set VV containing f(x)f(x), there exists an open set UU containing xx with f(U)⊆Vf(U) \subseteq V; or for every subset AA, f(cl(A))⊆cl(f(A))f(\mathrm{cl}(A)) \subseteq \mathrm{cl}(f(A)) (the image of the closure sits inside the closure of the image). Each formulation captures a different facet of the same geometric fact: continuous maps do not introduce discontinuities.

Continuous maps compose: if f:X→Yf: X \to Y and g:Y→Zg: Y \to Z are both continuous, then g∘f:X→Zg \circ f: X \to Z is continuous. The identity map on any space is continuous. These two facts mean that topological spaces and continuous maps form a category Top\mathbf{Top}, with homeomorphisms — continuous bijections with continuous inverses — as isomorphisms. Two spaces are homeomorphic when they are topologically identical: every topological property shared by one is shared by the other.

Familiar examples of continuous maps include constant functions (which are always continuous), inclusion maps from subspaces, and projections from product spaces. Less obviously, every continuous map f:X→Yf: X \to Y induces a geometric morphism between the sheaf topoi Sh(X)\mathbf{Sh}(X) and Sh(Y)\mathbf{Sh}(Y): the direct image f∗f_* pushes sheaves forward, and its left adjoint f∗f^* pulls them back.
