2026-03-30 A triple (A, r, X) of a set of states, a set of attributes, and an evaluation function r: A × X → L, generalizing formal contexts and connecting to linear logic.

A Chu space (Barr 1979, developed by Pratt 1999) over a value set LL is a triple (A,r,X)(A, r, X) where AA is a set of states (or objects), XX is a set of attributes (or observations), and r:A×X→Lr: A \times X \to L is an evaluation function.

Pratt (1999) proved that a formal context (G,M,I)(G, M, I) is a Chu space over {0,1}\{0, 1\}, with GG as states, MM as attributes, and II as evaluation. Many-valued formal contexts are Chu spaces over the value set LL.

Chu spaces form a ∗*-autonomous category — the categorical structure underlying linear logic. Tensor in linear logic corresponds to the tensor product of Chu spaces.

Jakl (2025) proved the key connection: the Chu construction Chu(H,0)\mathrm{Chu}(H, 0) over a Heyting algebra HH is exactly the twist construction that produces bilattices. Under this identification:

  • Tensor (⊗\otimes) in linear logic = fusion (⊙\odot) in the twist product = extending
  • Linear implication (⊸\multimap) = twist-product implication (⇒\Rightarrow) = restricting

This means every bilattice-valued formal context is simultaneously a Chu space, a model of linear logic, and a concept lattice with Lambek calculus structure.
