Claude

2026-03-30 Anthropic's family of large language models — the AI system behind Claude Code, the developer API, and consumer products at claude.ai.
Defines Claude
Requires transformer

Claude is Anthropic’s family of large language models, built on the transformer architecture and trained using Constitutional AI and RLHF. Claude powers the consumer interface at claude.ai, the developer API, and Claude Code.

Each Claude generation ships in multiple tiers that trade capability for speed and cost:

  • Opus — highest capability, deepest reasoning, best for complex analysis and agentic tasks
  • Sonnet — balanced performance and cost, the default for most applications
  • Haiku — fastest and cheapest, optimized for high-throughput tasks

The current generation (as of early 2026) comprises Opus 4.6, Sonnet 4.6, and Haiku 4.5.

Claude models support text generation and analysis, vision (image understanding since Claude 3), structured tool use via the API, computer use (GUI interaction via screenshots and mouse/keyboard actions since Claude 3.5 v2), extended context windows up to 200K tokens (1M for some Opus configurations), and extended thinking for step-by-step reasoning on complex problems.

Access methods include the claude.ai web and mobile interface, the Claude API via Anthropic SDKs (Python, TypeScript), Claude Code as CLI and IDE extensions, and third-party platforms (Amazon Bedrock, Google Cloud Vertex AI, GitHub Copilot).

For the release history and evolution of the Claude model family, see Claude Model Releases.
