Use Git Basics

2025-12-31 Use basic git commands for status, staging, committing, and syncing. Use when making changes in a git repository.
  • Check status with git status before and after changes.
  • Stage relevant files with git add.
  • Commit changes with a descriptive message.
  • Pull before pushing when collaborating.
