llama.cpp

2026-03-08

llama.cpp is an open-source C/C++ implementation of large language model inference, originally built for Meta’s Llama models but now supporting most open-weight model architectures. It is the inference engine underneath Ollama.

llama.cpp was created to run LLMs on consumer hardware — particularly CPUs — without requiring Python, CUDA, or other heavy dependencies. It reads models in GGUF format and supports quantization from 2-bit to 8-bit, making it possible to run 7B-parameter models on machines with 8 GB of RAM.

Hardware support includes CPU (x86, ARM), NVIDIA GPU (CUDA), AMD GPU (ROCm), Apple Silicon (Metal), and Vulkan. An experimental backend for Qualcomm NPU via QNN is under development but not yet in the mainline.
