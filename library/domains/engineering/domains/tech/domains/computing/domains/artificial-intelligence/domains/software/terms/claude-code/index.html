Claude Code

2026-03-30 Anthropic's agentic coding tool — a CLI and IDE extension that reads, writes, and executes code autonomously using Claude models.

Claude Code is Anthropic’s agentic coding tool. It operates as an interactive agent that reads and searches codebases, edits files, executes shell commands, manages git workflows, and runs tests — using Claude models to reason about each step. Unlike chat-based code assistants that suggest snippets for a human to paste, Claude Code acts directly on the filesystem and shell, making it an agent rather than an advisor.

Claude Code is available as a CLI (the claude command, installed via npm), a VS Code extension, a JetBrains extension, a desktop app (Mac and Windows), and a web app (claude.ai/code). All interfaces run the same underlying agent engine. See IDE integration for the editor extensions.

The agent’s built-in tools include Read, Edit, Write (file operations), Bash (shell execution), Glob and Grep (search), Agent (spawn subagents for parallel or isolated work), WebSearch and WebFetch (web retrieval), and TodoWrite (task tracking).

Claude Code is extensible through four mechanisms that operate at different levels: CLAUDE.md shapes the model’s reasoning via project instructions loaded at session start; slash commands and skills provide task-specific instructions invoked on demand; hooks execute shell commands deterministically on tool-call lifecycle events; and MCP servers connect external tools and data via the Model Context Protocol. For how these extension points relate and when to use each, see Extension Architecture in Claude Code.

A permission system governs which tools execute automatically and which require user approval, defaulting to confirmation for destructive or externally visible actions. Each conversation manages its finite context window through automatic compaction — summarizing earlier messages to maintain coherence across long sessions.

For the development history and adoption of Claude Code, see Claude Code Development and Adoption.
