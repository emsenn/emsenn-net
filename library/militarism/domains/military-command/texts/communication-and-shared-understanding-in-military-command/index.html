Communication and Shared Understanding in Military Command

2026-03-08 by claude
Abstract

A research text on how military command builds shared understanding through intent, liaison, and common operating pictures.

<p>Military command has a persistent communication problem. A force has to create shared understanding across dispersed people, unequal access to information, and rapidly changing conditions without assuming that every participant can see the same thing at the same time [@jp3_33; @adaptivec22025].</p> <h2 id="intent-as-communicative-compression">Intent as communicative compression</h2> <p>The Army&rsquo;s 2019 mission command doctrine rollout is useful because it shows how the institution frames mission command and command and control as a problem of shared understanding rather than only obedience <a href="#ref-cacmissioncommand2019" class="citation">[cacmissioncommand2019]</a>. In that setting, <a href="../terms/commanders-intent.md" class="link-internal">commander&rsquo;s intent</a> matters because it compresses purpose so subordinate action can remain coherent when detailed control breaks down.</p> <h2 id="liaison-as-organizational-translation">Liaison as organizational translation</h2> <p>The second device is the <a href="../terms/liaison-officer.md" class="link-internal">liaison officer</a>. Joint doctrine treats liaison as essential to coordination because one headquarters rarely understands another by default <a href="#ref-jp3_33" class="citation">[jp3_33]</a>. Liaison is therefore a translation function. It carries context, priorities, and constraints across organizational seams.</p> <h2 id="shared-pictures-and-their-limits">Shared pictures and their limits</h2> <p>The third device is the <a href="../terms/common-operating-picture.md" class="link-internal">common operating picture</a>. Army writing on the COP and later adaptive command-and-control work is useful because it shows that a shared picture is never just a display problem [@cop2012; @adaptivec22025]. A shared picture only helps when a force also shares categories, timing, and judgment about what matters.</p> <h2 id="communication-as-infrastructure">Communication as infrastructure</h2> <p>Taken together, intent, liaison, and COP are not separate conveniences. They are parts of the communication infrastructure of command. Intent compresses purpose. Liaison translates between organizations. COPs align attention around a common frame.</p> <h2 id="sources">Sources</h2> <ul> <li><a href="https://www.army.mil/article/225414/combined_arms_center_launches_new_mission_command_doctrine" class="link-external">Combined Arms Center launches new mission command doctrine</a></li> <li><a href="https://www.jcs.mil/Portals/36/Documents/Doctrine/pubs/jp3_33.pdf" class="link-external">Joint Publication 3-33: <em>Joint Task Force Headquarters</em></a></li> <li><a href="https://www.army.mil/article/92364/" class="link-external">U.S. Army&rsquo;s common operating picture tool continues to evolve</a></li> <li><a href="https://www.army.mil/article/286205/adaptive_c2_modernizing_army_command_and_control" class="link-external">Adaptive C2: Modernizing Army Command and Control</a></li> </ul>
