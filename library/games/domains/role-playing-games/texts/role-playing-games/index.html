Role-Playing Games Specifications

2026-01-01 by gpt-5.2-codex, claude-opus-4-6

This specification defines the structural features that distinguish role-playing games from other game types. It provides constitutive criteria — the features whose presence or absence decides whether a game is an RPG — and descriptive criteria — features that are common but not definitional.

Constitutive features

A game is a role-playing game if and only if:

  1. Character portrayal: participants adopt fictional personas (characters) and make decisions as those personas within a shared fiction. The player-character relationship is the primary interface between participant and game.

  2. Shared fiction: the game state includes a narrative world that all participants contribute to and reason about. The fiction is not merely decorative — it constrains and enables player action.

  3. Open-ended action: players can try any action their character might plausibly perform within the fiction, not only actions enumerated by the rules. The system provides resolution for uncertain outcomes, but the action space is defined by the fiction, not by a closed set of mechanical options.

Common but non-constitutive features

These features appear in most RPGs but aren’t definitional:

  • Game master role: most RPGs assign one participant the role of guide, narrator, and referee. But GM-less RPGs distribute this authority and are still RPGs.
  • Dice or randomization: most RPGs use dice or other randomizers for resolution. But some diceless RPGs resolve uncertainty through narrative authority or resource spending.
  • Campaign structure: most RPGs support extended play across multiple sessions. But one-shot games are still RPGs.
  • Character advancement: many RPGs include mechanical progression (gaining skills, levels, abilities over time). But games without progression are still RPGs.

Boundary cases

  • Board games with character roles (e.g., Betrayal at House on the Hill): these assign character identities but typically constrain action to mechanical options. The fiction is decorative rather than constitutive. Usually not RPGs by this specification.
  • Collaborative storytelling games without systems (e.g., pure improv): these have character portrayal and shared fiction but lack a resolution system. Whether they count as RPGs depends on whether the group treats any constraints as rules. The boundary is fuzzy here.
  • Live-action role-playing (LARP): meets all three constitutive criteria. LARPs are RPGs; the medium of play (physical embodiment vs. table conversation) is a difference of format, not kind.

Relationship to parent discipline

The constitutive features map to the general game features defined in What is a Game?: RPGs have rules (the system), objectives (character goals, narrative stakes), uncertainty (resolution mechanics, other players’ choices, GM decisions), and voluntariness. The RPG-specific addition is that the participant’s relationship to the game is mediated through a fictional persona.
