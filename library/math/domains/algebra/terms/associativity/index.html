2026-03-30 The property that grouping does not matter: (a · b) · c = a · (b · c). The first axiom of semigroups, monoids, groups, and categories.

A binary operation ∗\ast is associative if grouping does not matter:

(a∗b)∗c=a∗(b∗c)(a \ast b) \ast c = a \ast (b \ast c)

for all aa, bb, cc. When an operation is associative, you can write a∗b∗ca \ast b \ast c without parentheses — the result is the same either way.

Addition is associative: (2+3)+4=2+(3+4)=9(2 + 3) + 4 = 2 + (3 + 4) = 9. Function composition is associative: (h∘g)∘f=h∘(g∘f)(h \circ g) \circ f = h \circ (g \circ f). Subtraction is not: (7−3)−1=3(7 - 3) - 1 = 3 but 7−(3−1)=57 - (3 - 1) = 5.

Associativity is the first axiom of nearly every algebraic structure: semigroups, monoids, groups, rings, categories, residuated lattices. It is what makes sequential combination well-defined — you can chain operations without ambiguity.
