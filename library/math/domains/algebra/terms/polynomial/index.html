Polynomial

2026-03-05

A polynomial over a ring RR is a formal expression a0+a1x+a2x2+⋯+anxna_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n where ai∈Ra_i \in R and xx is an indeterminate. The set of all such expressions, with the usual addition and multiplication rules, forms the polynomial ring R[x]R[x].

If RR is a commutative ring, so is R[x]R[x]. If RR is an integral domain, so is R[x]R[x]. Polynomial rings are the basic tool for constructing new rings from old ones and underpin algebraic geometry, where the zeros of polynomials define geometric objects.
