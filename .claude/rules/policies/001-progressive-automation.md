Make actions accomplishable by less capable agents over time.

Trajectory: learn → document → curricula → skills → scripts → formalization.

When you finish a task, encode the repeatable parts as skills or scripts.
Prefer deterministic scripts over inference-based skills where the task
is well-understood. Design skills with enough detail that a less capable
model could follow them.
